# Jobraker Launch Storyboard

## Overview

The video opens with job-search pain and a visible enemy: ATS robots swallowing resumes. Jobraker then arrives as a counter-agent command center where slash commands, proof graphs, direct outreach, and Agent 50 automation turn the search into agent work.

## Scene List

| Scene | Time | Composition | Creative Direction |
|---|---:|---|---|
| 0 | 0.0-5.0 | `cold-open` | Tiny resume PDF attacked by rejection cards and pulled into an ATS vortex. Hook text glitches into the counter-agent reveal. |
| 1 | 5.0-11.0 | `agent-command` | Dark chat/terminal UI snaps open. Slash commands cascade, user asks for winnable roles, and a green agent orb forms. |
| 2 | 11.0-19.0 | `fit-intelligence` | Split-screen proof graph maps candidate evidence to job requirements. Fit score resolves from 42% to 87%. |
| 3 | 19.0-27.0 | `direct-apply` | Chat command `/directapply @Linear-style-company` expands into a plugin card with human-review controls. |
| 4 | 27.0-35.0 | `agent50` | Agent 50 orb traces through job board, career page, email draft, tracker, browser form, and notification panels. |
| 5 | 35.0-42.0 | `success-cta` | Scene 0 chaos reverses into positive outcomes. Jobraker lockup and CTA hold for the final second. |
